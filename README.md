# FUTURE_CS_02

## Phishing Email Analysis

### Indicators of Phishing

1. Fake sender email: support@paypaI-security.com (uses a capital "I" instead of a lowercase "l").
2. Urgent subject line: "Urgent! Verify Your Account".
3. Generic greeting: "Dear Customer".
4. Suspicious link: http://paypal-login-security.com (not the official PayPal website).
5. Threatens account closure within 24 hours.
6. Tries to create panic and pressure the user into clicking the link.

### Conclusion

This email is a phishing attempt because it uses urgency, a fake sender address, a suspicious URL, and threats to trick users into revealing their account information.
